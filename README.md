# Tarjeta de Navidad (Sara Guevara)

## Cómo usar
1. Abre `index.html` (doble click) o súbelo a tu hosting (GitHub Pages, Netlify, etc.).
2. Toca la tarjeta para abrirla.

## Personalizar nombres por URL
- **Para**: `?para=Sara%20Guevara`
- **De**: `?de=Jino`
- Abrir directo: `?open=1`

Ejemplo:
- `index.html?para=Sara%20Guevara&de=Jino&open=1`

## Archivos
- `index.html` estructura
- `styles.css` estilos (responsive)
- `script.js` interacción + nieve/confeti + typewriter
- `assets/navidad.jpg` imagen de portada
- `assets/regalos.jpg` textura de fondo

Si quieres cambiar las imágenes, reemplaza los JPG en `assets/` manteniendo el mismo nombre.
